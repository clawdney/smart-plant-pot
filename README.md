# 🌱 Smart Plant Pot

IoT plant pot with soil sensor, self-irrigation, and web dashboard.

## Features

- 📊 **Real-time soil moisture monitoring**
- 💦 **Automatic irrigation** when soil is too dry
- ⏰ **Scheduled watering** (set time daily)
- 📱 **Web dashboard** - access from anywhere
- 🔧 **Manual override** - start/stop pump anytime
- 📈 **Event history** - track all watering events

## Hardware

- ESP8266 (NodeMCU)
- Soil moisture sensor (analog)
- 5V Relay module
- Water pump (5V-12V)
- Power supply

### Circuit Diagram

```
ESP8266
├── A0 ────── Soil Sensor (analog)
├── D1 ────── Relay IN (water pump)
├── 3.3V ──── Soil Sensor VCC
├── GND ───── Soil Sensor GND
└── GND ───── Relay GND

5V Power Supply
├── + ──────── Pump + | Relay NO
├── ───────── Relay COM | Pump +
└── ───────── Relay NC (not used)
```

## Software

### ESP8266 Firmware

Upload `esp8266/firmware/firmware.ino` using Arduino IDE.

**Libraries required:**
- PubSubClient
- ArduinoJson
- ESP8266WiFi

### Dashboard Server

```bash
cd dashboard
pip install flask
python app.py
```

The dashboard runs on port 5000 and creates a SQLite database for storing events and settings.

## Access from Internet

### Option 1: OpenClaw Tunnel (if running)

Access at: `http://<your-openclaw-url>:5000`

Connect via NordVPN mesh or similar to access local network.

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/devices/register` | Register device |
| POST | `/api/devices/<id>/event` | Send event |
| POST | `/api/devices/<id>/status` | Update status |
| GET | `/api/devices/<id>/events` | Get event history |
| GET | `/api/devices/<id>/settings` | Get settings |
| POST | `/api/devices/<id>/command` | Send command |
| GET/POST | `/api/schedule` | Manage schedule |

## MQTT Topics

- Subscribe: `smart-plant-pot/command`
- Publish: `smart-plant-pot/status`

## Configuration

Edit `esp8266/firmware/firmware.ino` to set:
- WiFi credentials
- MQTT server IP
- API server URL
- Pin assignments
